## üìù Homework Assignment

### Mini-Project: Manufacturing Quality Analysis

**Objective:** Apply your Python and ML skills to a real-world manufacturing dataset.

**Task:**
Build a complete data analysis and ML pipeline that:
1. Loads and explores a manufacturing defect dataset
2. Performs comprehensive EDA with visualizations
3. Preprocesses data (handling missing values, scaling, encoding)
4. Trains multiple classification models
5. Evaluates and compares model performance
6. Generates insights and recommendations

**Dataset:** Provided in `datasets/manufacturing/quality_data.csv`

**Deliverables:**
- Jupyter notebook with complete analysis
- README explaining your approach
- Saved model (best performing)
- Report with findings (markdown or PDF)

**Rubric:**
- **Data Exploration (20%):** Thorough EDA with insights
- **Preprocessing (20%):** Proper handling of data issues
- **Modeling (30%):** Multiple models, proper evaluation
- **Code Quality (15%):** Clean, commented, reproducible
- **Insights (15%):** Clear recommendations

**Submission:** Create `homework/week-01-02/` directory

## üîç Common Pitfalls

### Python
- ‚ùå Mutable default arguments
- ‚ùå Not using list comprehensions
- ‚ùå Ignoring exceptions
- ‚úÖ Write Pythonic, readable code

### NumPy/Pandas
- ‚ùå Using loops instead of vectorization
- ‚ùå Not handling missing data
- ‚ùå Memory inefficiency
- ‚úÖ Leverage built-in functions

### Machine Learning
- ‚ùå Not splitting train/test data
- ‚ùå Data leakage
- ‚ùå Overfitting without regularization
- ‚úÖ Follow ML best practices
